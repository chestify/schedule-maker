# ScheduleMaker – Offline Class Weekly Schedule Maker

ScheduleMaker is a Windows desktop app designed for teachers and tutors who need a simple, reliable way to plan recurring weekly lessons.

## Why ScheduleMaker?

- Works completely offline after activation  
- No login required  
- No subscription  
- One-time purchase  
- Designed specifically for educators  

## Features

- Weekly class schedule planning  
- Student and session management  
- Recurring weekly structure  
- Export and import backups  
- Clean, distraction-free interface
